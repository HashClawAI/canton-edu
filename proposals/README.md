# Canton Development Fund — proposal files

## Submitting

1. Fork [canton-foundation/canton-dev-fund](https://github.com/canton-foundation/canton-dev-fund) (e.g. [HashClawAI/canton-dev-fund](https://github.com/HashClawAI/canton-dev-fund)).
2. Copy `proposals/open-canton-edu.md` from this repo into your fork as `proposals/open-canton-edu.md`.
3. Open a PR to **upstream** `canton-foundation/canton-dev-fund`.
4. Paste **`PR_BODY_OPEN_CANTON_EDU.md`** into the PR description. Match [`.github/pull_request_template.md`](https://github.com/canton-foundation/canton-dev-fund/blob/main/.github/pull_request_template.md) on upstream (headings: **Development Fund Proposal Submission**, **Proposal file**, **Summary**, **Checklist**, **Notes for reviewers**).
