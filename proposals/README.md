# Canton Development Fund — proposal drafts

Official program repo: [canton-foundation/canton-dev-fund](https://github.com/canton-foundation/canton-dev-fund).

To submit: fork that repo, add the proposal file under `proposals/` on a branch, open a PR titled `Proposal: <Project Name>`, and fill in the GitHub PR body using the checklist in `.github/pull_request_template.md`.

The Markdown file in this folder is ready to copy into your fork. Adjust **Champion contact** (e.g. GitHub @), **dates**, **CC totals**, and **event cities / format** after Committee feedback. The current draft centers **Asia-Pacific developer community**, **≥2 developer salons**, **multilingual + i18n**, **site launch**, and **12-month maintenance**, with an **illustrative 120,000 CC** split (see §6.1 in the proposal on funding realism).

Proposal sections mirror the [program README](https://github.com/canton-foundation/canton-dev-fund): **Objective and scope**, **Technical approach**, **Architectural alignment**, **Milestones and deliverables**, **Acceptance criteria**, **Funding request and milestone breakdown**, plus adoption plan, capability evidence, and open-source outputs.
