# Canton Development Fund — proposal drafts

Official program repo: [canton-foundation/canton-dev-fund](https://github.com/canton-foundation/canton-dev-fund).

To submit: fork that repo, add the proposal file under `proposals/` on a branch, open a PR titled `Proposal: <Project Name>`, and fill in the GitHub PR body using the checklist in `.github/pull_request_template.md`.

The Markdown file in this folder is ready to copy into your fork (adjust author, champion, CC amounts, and dates before opening the PR).
